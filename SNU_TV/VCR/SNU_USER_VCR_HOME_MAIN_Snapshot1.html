This is a legacy SNU feature meant to emulate a VCR. The differences between the DVR and the VCR includes

* Videos can be no longer than 14400 seconds (240 minutes) (4 hours) in length, however the default catalog doesn't exceed 1 hour.

* Only older content can be played

* Home movies are very relevant on the VCR mode

* You can upload video files and play them here

File size (catalog) 2 Gigabytes
